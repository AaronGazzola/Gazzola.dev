Add an item to the top of the site map items with "Add ...", clicking this should reveal a text input. on the right side of the text input is a "/" and a plus icon on the right (like this "/ + "). Clicking the plus icon will add another input. On the far end of the inputs is a save icon button. Clicking the save icon button or hitting enter from an input will add a folder and page.tsx file to the AppStructure
