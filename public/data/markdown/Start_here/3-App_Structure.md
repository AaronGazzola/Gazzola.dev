# App Structure

The app directory provides a site map, a UI layout heirarchy, and a data management strategy.

- `page.tsx` files represent pages in the site map
- `layout.tsx` files wrap all descendant pages
- Files ending in `.stores.tsx` contain Zustand stores
- Files ending in `.hooks.tsx` contain React-Query hooks
- Files ending in `.types.tsx` contain types
- Files ending in `.actions.tsx` contain Next.js server actions
