## Setup Instructions for Windsurf

### Step 1: Install and Setup

1. Download and install [Windsurf](https://codeium.com/windsurf)
2. Install [Git](https://git-scm.com/) if not already installed
3. Open Windsurf and sign into your GitHub account when prompted

### Step 2: Create and Clone Repository

1. Create a new repository on [GitHub](https://github.com)
2. In Windsurf, press `Ctrl+Shift+P` (Windows/Linux) or `⇧⌘P` (Mac) to open the Command Palette
3. Type `Git: Clone` and select it
4. Choose **Clone from GitHub** and sign in if prompted
5. Select your repository from the dropdown
6. Choose a local folder for the project and click **Open**

**For Private Repositories (SSH):**
- First, generate an SSH key: Open a terminal and run `ssh-keygen -t rsa -b 4096 -C "your_email@example.com"`
- Add the public key to GitHub: Go to GitHub Settings → SSH and GPG keys → New SSH key
- When cloning, use the SSH URL format: `git@github.com:username/repo.git`

### Step 3: Initialize Your Next.js Application

Ask the AI to set up your project:
```
Please initialize a Next.js web app in the root of this repository
with all latest defaults accepted.
```

### Step 4: Add Your Roadmap Files

1. Download your roadmap directory from the sidebar in Gazzola.dev
2. Drag and drop your downloaded roadmap files into the project root

### Step 5: Commit and Push to GitHub

Ask the AI to commit and push your changes:
```
Please commit all changes with the message "Add project roadmap"
and push to GitHub
```

### Step 6: Start Building with AI

Direct the AI to your roadmap:
```
Please read the roadmap/README.md file and help me implement
the features and requirements described there.
```
