## Development

### Running the Application Locally

Start the development server:

```bash
npm run dev
```

Open your web browser and visit [http://localhost:3000](http://localhost:3000)

The application will automatically reload when you make changes to the code.

### Available Commands

- `npm run dev` - Starts the development server
- `npm run build` - Creates an optimized production build
- `npm run start` - Runs the production build locally
- `npm run lint` - Checks code for potential issues
<!-- /option-1 -->

<!-- option-2 -->
- `npm run test` - Runs all tests
- `npm run test:e2e` - Runs end-to-end tests with Playwright
<!-- /option-2 -->

<!-- option-3 -->
- `npm run cypress:open` - Opens Cypress test runner
- `npm run cypress:run` - Runs Cypress tests in headless mode
<!-- /option-3 -->

<!-- option-4 -->
- `npx prisma studio` - Opens a visual database editor
- `npx prisma migrate dev` - Creates and applies database migrations
- `npx prisma generate` - Generates Prisma client types
